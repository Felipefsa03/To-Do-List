en-Us: During the process, I worked on important concepts such as componentization, clean architecture, and data flow organization, which contributed to a more scalable and easily maintainable application. I implemented navigation using React Router, simulating real-world SPA application scenarios, and created HTTP request emulations to practice asynchronous flow and state handling such as loading and error. I also used Local Storage to ensure data persistence, guaranteeing a smoother user experience. The use of TypeScript was essential to bring security to the development, ensuring strong typing and reducing potential failures during code evolution. Furthermore, I used pnpm as a package manager to achieve greater efficiency in the development environment.

Pt-Br: Durante o processo, trabalhei conceitos importantes como componentização, arquitetura limpa e organização de fluxo de dados, o que contribuiu para uma aplicação mais escalável e de fácil manutenção. Implementei navegação utilizando React Router, simulando cenários reais de aplicações SPA, além de criar emulações de requisições HTTP para exercitar o fluxo assíncrono e o tratamento de estados como carregamento e erro. Também utilizei Local Storage para garantir persistência dos dados, garantindo uma experiência mais fluida ao usuário. O uso de TypeScript foi essencial para trazer segurança ao desenvolvimento, garantindo tipagem forte e reduzindo possíveis falhas durante a evolução do código. Além disso, utilizei o pnpm como gerenciador de pacotes para obter mais eficiência no ambiente de desenvolvimento.
